# py-matrix

py-matrix is a third-party Python library that encapsulates a large number of operations on matrices and vectors. 

## Installation

> install

## Usage

> usage: python

## Independencies

- **Numpy**, `version: 1.24.4`
- **Pandas**, `version: 2.0.3`
